    1) A VPC
    2) 2 Private subnets
     3) 2 Public subnets
     4)  2 eip
     5)  2 NAT gateway
     6)  An internet gateway
     7)  Am IAM role and policy attached to the EKS cluster and an IAM role and policy attached to the node group(worker node)
      8)  An EKS Cluster
      9)  The node group(worker nodes)
